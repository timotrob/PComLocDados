## Descriminação das Colunas nos arquivos

### Bts.csv

| Coluna  | Descrição |
| ------------- | ------------- |
| grupo  | Localização do setor  |

| cch  | Canal de Controle BTS  |
| azimuth | Azimute da Antena da BTS  |
| RssiId |Identificador no arquivo de medições (treino/test) |
